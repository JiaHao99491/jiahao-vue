## 使用方式

### 全局注册

```js
import 'jiahao-vue/lib.css'
import { createJiaHaoVue } from 'jiahao-vue'
app.use(createJiaHaoVue())
```

### 组件中使用

```js
import 'jiahao-vue/lib.css'
import { JiaHaoButton } from 'jiahao-vue'
```
